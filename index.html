Things I Do

Vietnam flag Led circuit model

This project involves designing and building an LED-based replica of the Vietnamese flag. Red LEDs will be used for the flag's background, and yellow LEDs will create the five-pointed star. The circuit will be constructed on a PCB and can be controlled using a logic circuit. This project is intended to pay tribute to the Vietnamese national flag and can be used for decorative or display purposes.

Timer clock using 7SEG LED circuit model

This project aims to design and build a simple, unique digital clock using seven-segment LEDs. A microcontroller will be used to control the LEDs, generating numbers from 0 to 9 to display the accurate time.

A 2D Java Adventure-Action Game

This project focuses on developing a 2D game featuring adventure and action elements. Players will embark on a quest to defeat monsters, gain experience points to level up, and use in-game currency to purchase items. Throughout their journey, players must find keys to unlock dungeons, solve puzzles, and engage in intense boss battles.
